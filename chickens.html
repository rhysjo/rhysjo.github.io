Standard Elements:

  • Comment - "<!-- This is a comment -->"
    • The comment is ignore by the browser when rendering
  • Bold - "<b> This is a bold tag </b>"
    • Makes the specified text appear bold
  • Italics - "<i> This is an italics tag </i>"
    • Makes the specified text appear italic
  • Paragraph - "<p> This is a paragraph tag </p>"
    • Adds a paragraph element to the page
  • Unordered List (Bullet Points) - "<ul> This is an unordered list tag </ul>"
    • Starts a bullet pointed list. Lists can be nested
  • Ordered List (Numbered) - "<ol> This is an ordered list item tag </ol>"
    • Starts a numbered - or ordered - list. Lists can be nested
  • List Item - "<li> This is an unordered list item tag </li>"
    • Adds a new list item to the list. Must be nested in a <ul> or <ol> tag
  • Nested List Items
    • See above links for more info
  • Anchor - "<a href="https://www.google.co.uk/"> This is an anchor/hyperlink </a>"
    • href: Hypertext REFerence
  • Image - "<img src="" alt="">" This is an image tag
    • <img> doesn't use a closing tag
  • Div - "<div> This is a div tag </div>"
    • The div is a generic container to hold/group elements together. It is a block level element
  • Span - "<span> This is a span tag </span>"
    • The span is container for inline elements. CSS can be applied to alter appearance
  • Hr - "<hr> This is a hr tag"
    • The hr is a line separator. See after the image below. It doesn't need a closing tag
  • Br - "<br> This is a br tag"
    • The br is a line break. It doesn't need a closing tag
  • Supsuperscript - "<sup> This is a sup tag </sup>"
    • The sup tag allows superscript text
  • Subsubscript - "<sub> This is a sub tag </sub>"
    • The sup tag allows superscript text

Semantic Elements

Semantic elements offer no new functionality compared to the elements already available. They offer better readability and structure within the HTML code. They're also beneficial for screen reader users.

HTML Boilerplate

There's a standard boilerplate for any new HTML file. Can be auto-generated in VS Code by typing ! and pressing TAB.

Boilerplate

<!DOCTYPE html> --> This is a flag to the browser to let it know it's viewing a HTML5 file
<html> --> Represents the top level of the document
<head> --> Contains (metadata) about the document i.e. scripts and style sheets
      <title>My First Page</title> --> Sets the page/tabs title name
</head>
<body> --> The content of the webpage begins here and there is only one body
      <!-- Content Goes Here -->
</body>
</html>

VS Code Shortcuts

  • ! then TAB: To generate HTML boilerplate
  • Shift + Alt + F: Format Document
  • Shift + Alt + Down: Duplicate current element
  • Ctrl + /: Comment line out
