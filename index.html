HTML Elements Documentation (MDN)

Standard Elements:

  • Comment - "<!-- This is a comment -->"
  • Bold - "<b> This is a bold tag </b>"
  • Italics - "<i> This is an italics tag </i>"
  • Paragraph - "<p> This is a paragraph tag </p>"
  • Anchor - "<a href="https://www.google.co.uk/"> This is an anchor/hyperlink </a>"
  • List Item - "<li> This is a list item </li>"



    • Heading Elements:

    Note: There should only be one h1 tag, there are multiple here for demonstration purposes.

    "<h1> This is a h1 tag </h1>"

    "<h2> This is a h2 tag </h1>"

    "<h3> This is a h3 tag </h1>"

    "<h4> This is a h4 tag </h1>"

    "<h5> This is a h5 tag </h1>"
    "<h6> This is a h6 tag </h1>"
